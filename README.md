# Social Maps App

A cool social mapping application where friends can share locations and connect in real-time.

## Features

- Interactive maps with custom markers
- Real-time location sharing
- Friend system and notifications
- User profiles and authentication
- Responsive design

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Leaflet for maps
- Socket.io for real-time features
- Zustand for state management
